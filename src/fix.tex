Let $c(A) = 1/2 - \delta$ 
then $c(x) > 1/2 + \delta$, $c(y) \leq 1/2 - \delta$, and $c(B) \geq 2\delta$.

\begin{lemma}
Let $S$ be any set and $C$ be the set produced by the algorithm 
before dropping the first element from $S$ then $f(C) \geq (1 - e^{-\frac{c(C)}{c(S)}})f(S)$.  
\end{lemma}
